# Twist Hero Migration

Fresh React + Node.js scaffold that recreates the Twist-style hero section using clean components and local assets (no Framer runtime dependencies).

## Run

Install dependencies:

```bash
npm install
```

Start the React client in development:

```bash
npm run dev:client
```

Build the client:

```bash
npm run build
```

Serve the built app with Node/Express:

```bash
npm start
```
